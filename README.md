# EDS-app
This is an early screening system for dementia. You can use our publicly available website to fill out the corresponding questionnaire and find out the type of dementia you may currently have

The four files "catboost_add_model.cbm", "catboost_cog_model.cbm", "X_train(μ,σ).xlsx", and "NACC数据_哑变量还原_特征筛选.xlsx" are essential scripts and data required for the system to operate. If you intend to run the current system, please place the system scripts and these files in the same directory.
